# Invetory Management System (FULLSTACK)
Sistema de gestion de ineventario con autenticación JWT, reportes automáticos, envíos de correos y actualizaciones en tiempo real con Socket.IO.

---

## Tecnologías principales
-Backend: Node.js + Express + Mongoose
-Frontend: Angular
-Socket.IO para tiempo real
-MongoDB
-Docker & docker-compose
-Postman (Pruebas)
